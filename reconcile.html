PXM-BM MT5 reconciliation

This is a reconciliation tool that should be used to reconcile the positions of Bermuda account of record BERMUDA_PXM and dealer account BRPXM_DEALER against their MT5 positions based on the MT5 logins, symbols and excluded groups.
Step 1: Go to Trading Accounts on MT5, make sure Login and Group columns are available, right click and export and save it to CSV file.
Step 2: Go to Positions on MT5, make sure Login, Symbol, Type and Volume columns are available, right click and export, select CSV file, click save but do not click Yes yet, just keep it available on your screen.
Step 3: Go to PXM Portal Position component, make sure there is no filters on, right click and export. As soon as you click export on PXM, click yes on MT5 to save positions.
Step 4: Go to PXM Real-time config > ConnectorRoute > right click and export the rules there.
Step 5: Input the CSV file exported from ConnectorRoute.
Step 6: Input PrimeXM positions exported from the Position component of the account module.
Step 7: Input MT5 Positions exported from MT5.
Step 8: Input MT5 Accounts exported from MT5

The CSV file should include connector, sym, subid1, and subid3 columns
The CSV file should include Account, Symbol, Base and Overflow columns
The CSV file should include Login, Symbol, Type, and Volume columns
The CSV file should include Login and Group columns
